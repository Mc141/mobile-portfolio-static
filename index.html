Work

Each project is a chapter—code written with purpose, design shaped by thought.

Chapter 1: Portfolio Digital Book

This portfolio itself is a chapter in my story. What you’re flipping through isn’t just a website—it’s a crafted artifact. Designed as an interactive book using Turn.js, it combines storytelling with code, demonstrating how I build not just function, but experience.

From layout to navigation, every detail was chosen to reflect care and intentionality. Note: The book experience is only available on desktop devices.

  • GitHub

Chapter 2: MCC Limpopo Website

This wasn’t just a web build—it was a business transformation. MCC Limpopo needed a digital presence that reflected their credibility. I built a custom site that presents their industrial offerings with clarity and structure.

It challenged me to think like both a developer and a user—designing for access, hierarchy, and reliability.

  • GitHub
  • Live Site

Chapter 3: Wellness Management App

Built with Java Swing and JavaDB, this desktop app simulates a wellness center’s internal operations—appointments, client feedback, counselor management, and more. It taught me to balance code structure with usability.

  • Built using MVC architecture and clean Java OOP principles
  • Fully interactive GUI with search, confirmation, and data validation
  • Local JavaDB database for persistence and retrieval
  • GitHub

Chapter 4: TheologyLens – AI Meets Scripture

TheologyLens is a unique project where artificial intelligence meets deep questions. It lets users ask scripture-related questions and returns meaningful passages—not by keyword, but by meaning.

  • Built with SentenceTransformer & FAISS for vector-based semantic search
  • Backend in Flask (Python), with a modular front end in React
  • Clustered passages reveal themes and relationships across scripture
  • GitHub

About

"Reason is the natural order of truth; but imagination is the organ of meaning."

I am M.C. Wolmarans, a software engineering student with a love for reasoned thought and crafted code. My work lies where logic meets imagination, where syntax serves ideas, and structure reflects purpose.

My focus is full-stack development with C#, Node.js, SQL, and React. I also spend time studying theology and philosophy, pursuing the deeper questions behind the tools we build and the systems we shape.

I believe that technology carries meaning. It reflects the values we embed in it, consciously or not. That is why I aim to write software with clarity, integrity, and a sense of responsibility for the world it helps create.
